# 📏 Feature Scaling

This folder contains my notes and Jupyter notebooks on **Feature Scaling**, an essential part of **Feature Engineering** in Machine Learning.

Feature scaling transforms numerical features to a common scale, helping many machine learning algorithms train faster, converge better, and produce more reliable results.
