# 🛠️ Feature Engineering

Welcome to my **Feature Engineering** learning journey!

This directory documents everything I learn about transforming raw data into meaningful features that improve the performance of Machine Learning models. Feature Engineering is one of the most important stages of the ML pipeline, directly influencing model accuracy, robustness, and generalization.

The notebooks in this folder focus on both the theoretical concepts and their practical implementation using Python and Scikit-learn.

## 📚 Topics Covered

### 📏 Feature Scaling
- Standardization (Z-Score Scaling)
- Min-Max Scaling
- Max Absolute Scaling
- Robust Scaling
- Choosing the right scaling technique

### 🔄 Feature Transformation
- Mathematical Transformations
- Log Transformation
- Reciprocal Transformation
- Square & Square Root Transformation
- Box-Cox Transformation
- Yeo-Johnson Transformation
- Power Transformations

### 🩹 Missing Value Imputation
- Handling Missing Numerical Values
- Handling Missing Categorical Values
- Mean, Median & Mode Imputation
- Random Sample Imputation
- KNN Imputation
- Multivariate Imputation
- Missing Indicator Technique
- End of Distribution Imputation

### 🏷️ Handling Categorical Features

- Label Encoding

- Count Encoding
- Target Encoding
- Binary Encoding
- Feature Hashing
- Rare Category Encoding

### 🔢 Handling Numerical Features

### 📈 Outlier Detection & Treatment
- Z-Score Method
