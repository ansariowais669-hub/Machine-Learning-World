# 🏷️ Encoding Data

This folder contains my notes and Jupyter notebook on **Encoding Categorical Variables**, an essential step in the Feature Engineering pipeline for Machine Learning.

Many Machine Learning algorithms work only with numerical data, making categorical data encoding a crucial preprocessing step before model training.

---

## 📘 Notebook

### 📓 Encoding_Categorical_Data.ipynb

In this notebook, I learned and implemented various techniques for encoding categorical features, including:

- 🔹 Label Encoding
- 🔹 Ordinal Encoding
- 🔹 One-Hot Encoding using Pandas
- 🔹 K-1 (Dummy Variable) One-Hot Encoding
- 🔹 One-Hot Encoding using Scikit-learn
- 🔹 One-Hot Encoding with Top Categories
- 🔹 Using `ColumnTransformer` for preprocessing pipelines

The notebook contains both the theoretical concepts and practical implementations of these encoding techniques on sample datasets.

---

## 🎯 Learning Outcomes

After completing this notebook, I learned:

- When categorical encoding is required
- The difference between nominal and ordinal categorical features
- When to use Label Encoding vs Ordinal Encoding
- How One-Hot Encoding works and why it is widely used
- The Dummy Variable Trap and K-1 Encoding
- How to implement encoding using **Pandas** and **Scikit-learn**
- How to build preprocessing pipelines using `ColumnTransformer`
