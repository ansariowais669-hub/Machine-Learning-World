# Handling Outliers

This folder contains notebooks demonstrating the fundamental concepts, detection techniques, and practical implementation of **outlier detection** in Machine Learning. It explains what outliers are, why they occur, their impact on data analysis, and how to identify them using widely adopted statistical methods.

## Topics Covered

- Introduction to Outliers
- Causes and Impact of Outliers
- Common Outlier Detection Techniques
- Detecting Outliers using **Z-Score**
- Detecting Outliers using **Interquartile Range (IQR)**
- Calculating Quartiles, IQR, and Outlier Bounds
- Identifying Outliers from a Sample Dataset

## Notebook

| Notebook | Description |
|----------|-------------|
| **OutlierAnalysis.ipynb** | Covers the theory behind outliers, explains their causes and effects, and demonstrates practical outlier detection using **Z-Score** and **IQR** methods with Python and NumPy. |
