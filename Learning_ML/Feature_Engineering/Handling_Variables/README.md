# 🧩 Handling Variables

Welcome to the **Handling_Variables** section of my **Feature Engineering** journey!

This folder contains my notes and implementations for handling special types of variables that commonly appear in real-world datasets. Proper feature engineering on these variables helps improve data quality and model performance.

---

## 📘 Notebooks Included

### 🔹 Handling_Date&Time_Variable.ipynb

In this notebook, I learned how to work with **date and time features** by extracting meaningful information from datetime columns.

#### Topics Covered

- Converting columns to `datetime` format
- Extracting year, month, day, and day of the week
- Identifying weekends
- Calculating elapsed time between dates
- Working with time features such as hours, minutes, and seconds
- Creating new numerical features from date and time data

---

### 🔹 Handling_Mixed_Variable.ipynb

In this notebook, I explored techniques for handling **mixed variables**, where numerical and categorical information are stored together in the same feature.

#### Topics Covered

- Identifying mixed variables
- Separating numerical and categorical components
- Extracting numeric values using string operations
- Extracting categorical labels
- Feature engineering on mixed variables
- Working with real-world examples such as Cabin, Ticket, and Passenger-related features
