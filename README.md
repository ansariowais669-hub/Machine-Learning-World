# 📌 My Learning Philosophy

I believe that Machine Learning is best learned by **building real projects**, **participating in competitions**, and **learning from mistakes**.

One thing I've realized early is that every model can be improved. Sometimes the biggest lessons come from decisions that didn't produce the best results. Instead of seeing those as failures, I see them as valuable learning experiences that help me make better decisions in future projects.

My goal isn't just to train models—it's to understand *why* they work and continuously improve my problem-solving skills.
